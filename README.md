# Athena

Athena is a UCI chess engine that can be used along with a chess GUI that
supports the UCI protocol.

Athena can be used to analyze chess games and to find the best move. It is
recommended the use of a chess GUI that supports the UCI (Universal Chess
Interface), the protocol Athena uses to communicate with other chess programs.
A chess GUI can display the information generated by a chess engine in a more
user friendly way, and it is also possible to play against the engine, where the
best move found is played by the GUI.

## How to compile

Clang is the default but Athena can probably be compiled with any other
compiler. To compile just execute `make` and install with `make install`. You
can edit the config.mk file to tune for your CPU. Define any of the following
macros to select specific instruction set extensions:

* USE_BMI
* USE_BMI2
* USE_POPCNT

For now only x86-64 is supported and you also need to define the macro ARCH_x64
to use any of the above extensions.
